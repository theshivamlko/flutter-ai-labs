## ğŸš€ Getting Started

### Prerequisites

- Flutter SDK (3.0 or higher)
- Dart SDK (3.0 or higher)
- An API key from your chosen AI provider (OpenAI, Anthropic, Google, etc.)

### Quick Start

1. **Clone the repository**

    ```bash
    git clone https://github.com/theshivamlko/flutter-ai-labs.git
    ```

2. **Navigate to the desired project directory**

    ```bash
    cd flutter-ai-labs/starter_ai_agents/chatgpt_clone
    ```

3. **Install Flutter dependencies**

    ```bash
    flutter pub get
    ```

4. **Set up your API keys**

    Create a `.env` file in the project root:
    ```env
    OPENAI_API_KEY=your_api_key_here
    ANTHROPIC_API_KEY=your_api_key_here
    GOOGLE_AI_API_KEY=your_api_key_here
    ```

5. **Run the app**

    ```bash
    # For mobile
    flutter run
    
    # For web
    flutter run -d chrome
    
    # For desktop
    flutter run -d macos  # or windows/linux
    ```

6. **Follow the project-specific instructions** in each project's `README.md` file for additional setup and configuration.
